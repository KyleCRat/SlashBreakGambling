# Changelog

All notable changes to Slash Break Gambling will be documented in this file.

## [12.0.1-3] - 2026-04-01

### Added
- Per-character stats tracking — winnings and losses recorded automatically for all addon users
- Stats UI panel with top 10 winners and top 10 losers
- Report Top 3 button — announces top 3 winners and losers to party/raid chat
- Report All button — announces all stats to party/raid chat in batched messages
- Stats slash commands: `/sbg stats`, `/sbg stats <name>`, `/sbg stats add <name> <amount>`, `/sbg stats rm <name>`, `/sbg stats reset`
- Stats sync via addon messages — all clients update when a game ends

## [12.0.1-2] - 2026-02-20

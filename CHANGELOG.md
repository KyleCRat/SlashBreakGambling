# Changelog

All notable changes to Slash Break Gambling will be documented in this file.

## [12.0.1-6] - 2026-04-09

### Changed
- Statistics panel now displays all players in a single ranked list from highest to lowest net gold
- Removed separate "Winners" and "Losers" section headers
- Top 3 winners are colored gold, silver, and bronze; remaining winners are green; losers are red
- Added a divider line between positive and negative entries
- Player names now display without the realm suffix

## [12.0.1-5] - 2026-04-07

### Fixed
- Fixed addon window incorrectly opening during ready checks and pull timers

## [12.0.1-4] - 2026-04-07

### Added
- Guild Cut option — set a percentage (0–100%) of winnings that goes to the guild
- Cycle through presets (0%, 5%, 10%, 25%, 50%, 75%) with left click, right click to reset to 0%
- Custom guild cut values via text input
- When guild cut is active, game results announce the winner/guild split in a single message
